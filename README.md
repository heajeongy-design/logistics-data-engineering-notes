# Logistics Data Engineering Notes

이 저장소에서는 단순히 기사를 요약하는 데 그치지 않고,  
산업 이슈를 데이터 구조, KPI, 이벤트 모델링, 운영 가시성 관점에서 해석하는 연습을 기록합니다.

## Repository Goals

- 물류 및 공급망 관련 기사 분석
- 데이터 엔지니어 관점의 해석 정리
- KPI 및 데이터 모델 아이디어 문서화
- 향후 대시보드/SQL/프로젝트로 확장
